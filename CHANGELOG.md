# Changelog

All notable changes to the Google ADK Cookbook.

The format loosely follows [Keep a Changelog](https://keepachangelog.com/).

## [0.1.0] — 2026-04-22

### Added

- Initial public release of the cookbook.
- Nineteen chapters covering introduction, getting started, core
  concepts, agent types, tools, skills, multimodal & live, computer
  use, deep research, multi-agent systems, memory patterns,
  observability, evaluation, deployment, safety, cost and latency,
  interop, comparisons, case studies, and a chapter on ADK as a
  harness platform.
- Twenty runnable example projects under `examples/`, each with a
  self-contained README and requirements.
- Patterns library (`patterns/`) with six production patterns.
- Cheatsheets (`cheatsheets/`) for imports, CLI, models, state
  prefixes, and services.
- Reference index linking authoritative sources.
- MkDocs Material configuration with Mermaid diagrams and editorial
  CSS.

### Versions referenced

- ADK Python: 1.31.1.
- Gemini models: `gemini-3.1-flash`, `gemini-3.1-pro`,
  `gemini-3.1-flash-lite`, `gemini-live-2.5-flash-native-audio`
  (Vertex — the only Live generation currently shipped),
  `gemini-2.5-flash-native-audio-preview-12-2025` (Gemini API Live),
  `gemini-2.5-computer-use-preview-10-2025`.
