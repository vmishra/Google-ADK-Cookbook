# 01 — hello-agent

A single `LlmAgent` with two function tools: rolls a die and checks
primality.

Covers: Chapter 1 (first agent), Chapter 3 (LLM agent).

## Run

```bash
pip install -r requirements.txt
python -m agent
```

Try:

```
> Roll a 20-sided die and tell me if the result is prime.
Rolled 13 — 13 is prime.
```
